# Intro to Translate

## Language Support

### [JavaScript](/javascript) • v0.0.1-0

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is a versatile language primarily used for client-side scripting in web development, but it is also increasingly used for server-side development and mobile app development.

### [Go](/go) • v0.0.1-0

Go, also known as Golang, is a statically typed, compiled programming language designed at Google. It aims to be simple, efficient, and reliable. Go is commonly used for building scalable and high-performance applications, particularly in areas such as cloud computing and microservices architecture.
